メニューカテゴリを表す列挙型

com.example.demo.domain.model.menu.MenuCategory

店舗で提供するメニュー項目の分類を定義する。

列挙定数名 displayName
APPETIZER "前菜"
MAIN_COURSE "メイン料理"
SIDE_DISH "サイドメニュー"
DRINK "ドリンク"
DESSERT "デザート"
フィールド フィールド型
displayName java.lang.String
メソッド 引数 戻り値型 説明
+ カテゴリの表示名を取得する String

注文ステータスを表す列挙型

com.example.demo.domain.model.order.OrderStatus

注文のライフサイクルにおける状態遷移を定義する。

状態遷移図

  PLACED → PREPARING → READY → SERVED
    ↓         ↓
  CANCELLED  CANCELLED
列挙定数名 displayName
PLACED "注文受付済み"
PREPARING "調理中"
READY "提供準備完了"
SERVED "提供済み"
CANCELLED "キャンセル"
フィールド フィールド型
displayName java.lang.String
メソッド 引数 戻り値型 説明
+ このステータスから指定されたステータスへの遷移が有効かどうかを判定する 注文ステータスを表す列挙型 boolean
+ ステータスの表示名を取得する String
