店舗ご注文システム API (1.0.0)

Download OpenAPI specification:

開発チーム: dev@example.com

実店舗で使用する注文管理システムのREST API仕様書。

概要

  • テーブル単位での注文管理
  • メニュー項目の参照
  • 注文ステータスのライフサイクル管理

アーキテクチャ

ヘキサゴナルアーキテクチャ + 簡易CQRS を採用。

ステータス遷移

PLACED → PREPARING → READY → SERVED
  ↓         ↓
CANCELLED  CANCELLED
