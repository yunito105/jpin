## 今日の講義

### 変数って何だろう
変数はメモリの一部を使ってデータを保存する仕組みです。
- メモリの領域にデータを保存できる
- 保存したデータを何度も再利用できる
- プログラムの中で値を参照・変更できる
- 変数に名前をつけることで、メモリのどこに何があるかを管理できる

### コンピュータの三大要素
コンピュータは3つの主要な要素で構成されています。キッチンで料理をする場面に例えると分かりやすいです：

1. **HDD/SSD（補助記憶装置）= 冷蔵庫**
   - 食材（プログラム・データ）を長期保存する場所
   - 電源が切れてもデータが残る（不揮発性記憶）
   - 容量が大きい

2. **CPU（中央処理装置）= シェフ**
   - プログラムの命令を実行する
   - 計算や制御を担当
   - 指示書の通りに作業を進める

3. **メモリ（主記憶装置）= まな板**
   - プログラム実行中のデータを一時保存
   - 冷蔵庫から取り出した食材を置く作業スペース
   - CPUが直接アクセスできる高速な記憶装置
   - 電源を切ると消える（揮発性記憶）

**プログラム実行の流れ：**
- HDDに保存されたプログラムはそのままでは動かない（冷蔵庫の中では調理できない）
- コンパイルして中間ファイルを作成
- コードをメモリにコピーして実行
- CPUがメモリ上の命令を実行し、データを処理する
